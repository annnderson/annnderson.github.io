Projeto 3 - Business Analytics Dashboard

Este projeto de Business Analytics tem como objetivo desenvolver um dashboard estratégico a partir de um modelo dimensional (Star Schema), permitindo a análise de indicadores financeiros, comerciais e operacionais.

A solução foi construída utilizando SQL e Power BI, cobrindo todo o ciclo analítico, desde a modelagem dos dados até a criação de KPIs e visualizações interativas, transformando dados brutos em insights estratégicos para tomada de decisão.

  • Construção de modelo dimensional (Star Schema)
  • Criação de KPIs financeiros e comerciais
  • Análise de receita, custo e margem
  • Identificação de padrões de performance por região, canal e produto
  • Otimização de consultas e performance em SQL

Problema de Negócio

  • Quais são os principais drivers de receita e margem?
  • Como está a performance de vendas ao longo do tempo?
  • Quais regiões e canais geram melhores resultados?
  • Quais produtos apresentam maior rentabilidade?
  • Onde estão as oportunidades de crescimento e otimização?
